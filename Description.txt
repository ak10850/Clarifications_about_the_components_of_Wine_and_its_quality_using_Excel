Design a Wine Quality dashboard to analyze the constitutes of Red and White Wine.
This Project includes:
1. Using of various formulae of an excel.
2. Created a User Control combo box for the wine categories i.e., Red and White Wine.
3. Created an interactive pie chart to show the percentage of compositions of different constitutions.
